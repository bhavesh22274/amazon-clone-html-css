# Amazon Clone (HTML & CSS)

This project is a frontend clone of the Amazon homepage created using only HTML and CSS.  
It focuses on layout, styling, and responsive design.

## Features
- Amazon-style header and navigation bar
- Search bar UI
- Product sections with images
- Footer with multiple columns
- Hover effects using CSS
- Clean and structured layout

## Technologies Used
- HTML5
- CSS3 (Flexbox, Box Model)

## Project Type
Frontend UI project (No JavaScript, No Backend)

## Purpose
This project was built to practice frontend development skills and understand real-world website layouts.

## Future Improvements
- Add JavaScript for interactivity
- Implement add-to-cart functionality
- Connect backend and database
